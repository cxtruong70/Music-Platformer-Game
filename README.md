# Music-Platformer-Game
A music platformer game written in processing (Build on Java) written by Calvin Truong. 

This version runs in 64-bit Windows.

When first trying to run the game, download the Processing IDE from processing.org.

Run the game by opening "NewStone" and before starting, make sure to import the sound library into the IDE.

Steps to import: Sketch Tab -> Import Library -> Add Library -> Search for "Sound" and import. 

Then, press just press play.
